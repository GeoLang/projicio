Features

📐 Projections

  • Web Mercator (EPSG:3857)
  • Transverse Mercator / UTM (zones 1–60)
  • Mercator (equatorial)
  • Lambert Conformal Conic (2 standard parallels)

🌍 Ellipsoids

  • WGS84 (GPS standard)
  • GRS80 (geodetic reference)
  • Clarke 1866 (NAD27)
  • International 1924
  • Unit sphere

🔄 Transform API

  • High-level EPSG code interface
  • Forward and inverse transforms
  • Thread-safe (Send + Sync)
  • All 60 UTM zones (N/S)
  • Sub-millimeter accuracy

🖥️ CLI

  • Transform coordinates between any supported CRS
  • EPSG code-based interface
  • Batch processing support
  • Human-readable output

Quick Start

use projicio_core::{Transform, Coord};

// Transform from WGS84 geographic to Web Mercator
let transform = Transform::new("EPSG:4326", "EPSG:3857").unwrap();
let result = transform.forward(&Coord::new(-73.9857, 40.7484)).unwrap();
// → (x: -8235851.2, y: 4975566.5) meters
# CLI usage
projicio transform --from EPSG:4326 --to EPSG:3857 -- -73.9857 40.7484
